# Papyrus

> A free, open-source, self-hostable PDF processing web app.

Merge, split, compress, convert, OCR, sign, redact, edit metadata, reorder/rotate pages, and more — through a fast browser UI backed by an async job pipeline. Privacy-first: zero-retention mode, no logging of document content, files purged on TTL.

## Status

Pre-alpha. Boilerplate scaffolding only — no features implemented yet.

## Stack

- **Web**: React 19, Vite, TypeScript, Tailwind CSS v4, shadcn/ui, TanStack Router + Query, Zustand, React Hook Form + Zod
- **API**: FastAPI, Python 3.13, Pydantic v2, SQLAlchemy 2.x async, asyncpg, Alembic
- **Workers**: Celery + Redis
- **Storage**: PostgreSQL 17, Redis 7, S3-compatible object store (MinIO locally)
- **Tooling**: uv (Python), pnpm (Node), Ruff, mypy, Biome, Vitest, pytest, Playwright

See [`CLAUDE.md`](./CLAUDE.md) for the full architecture, conventions, and code-style contract.

## Prerequisites

Install these natively on your dev machine — Docker is **not** used in development.

- Node 24+ (`.nvmrc`) and pnpm 9+
- Python 3.13 (`.python-version`) and [uv](https://github.com/astral-sh/uv) 0.5+
- PostgreSQL 17 — running locally on `:5432` with a `papyrus` role and database
- Redis 7 — running locally on `:6379`
- MinIO (or another S3-compatible store) — running locally on `:9000`

Docker / Kubernetes are only used for deployment. See [`infra/`](./infra/) for production images and manifests.

## Quick start

On a fresh machine, one command installs system packages (Postgres, Redis, Node, uv), downloads MinIO, creates the `papyrus` DB role + database, installs project deps, and applies migrations:

```bash
make setup
```

Run `bash scripts/setup.sh` directly if you want to inspect or pass `ASSUME_YES=1` / `SKIP_SYSTEM=1`.

After setup, run the data-plane and app processes (each in its own terminal):

```bash
bash scripts/run_minio.sh    # generated by setup; starts MinIO on :9000 (console :9001)
make api                     # http://localhost:8000
make worker
make web                     # http://localhost:5173
```

If you need to redo just the project-side install:

```bash
make install                 # creates .venv via uv, runs pnpm install
make db-upgrade
```
